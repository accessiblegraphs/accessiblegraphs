To access alternative presentations of this graph:
  1. Install the SAS Graphics Accelerator for Google Chrome from the Chrome Web Store.
  2. Follow these steps to enable SAS Graphics Accelerator to access local files:
    1. Press Alt to move focus to the Chrome menu.
    2. Use the arrow keys to navigate to More Tools > Extensions and then press Enter to open the Extensions page.
    3. Activate the Details button for SAS Graphics Accelerator and enable the "Allow access to file URLs" toggle button.
  3. Reopen this page in Google Chrome
  4. Activate the Accelerate button within the graph
